# Moatmt Landing Page (Static)

This is a static HTML/CSS/JS version of the Moatmt landing page.

## Structure

- **index.html**: Main landing page.
- **about.html**: About Us page.
- **blog.html**: Blog listing page.
- **blog-details.html**: Blog post detail template.
- **terms.html**: Terms and Conditions page.
- **public/**: Contains all assets (images, CSS, JS).
  - **assets/css/**: Stylesheets.
  - **assets/js/**: JavaScript files.
  - **images/**: Site images.

## Usage

You can serve this project using any static file server (e.g., Live Server in VS Code, Python SimpleHTTPServer, Apache, Nginx).

No build step is required.
